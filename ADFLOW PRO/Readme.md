

# AdFlow Pro
## Sponsored Listing Marketplace with Moderation, Payment Verification, Scheduling & Analytics

<img width="959" height="419" alt="1" src="https://github.com/user-attachments/assets/e7525105-6027-4910-8280-6bcbea294eee" />

## Overview

AdFlow Pro is a professional MERN Stack classified advertisement marketplace that provides a complete advertisement lifecycle from creation to publication.

The platform supports multiple user roles and ensures that advertisements are reviewed and verified before becoming publicly visible.

### Key Features

- JWT Authentication
- Role-Based Authorization
- Advertisement Moderation Workflow
- Payment Verification System
- Featured Advertisement Packages
- Analytics Dashboard
- Advertisement Ranking System
- Scheduled Publishing
- Advertisement Expiry Automation
- Responsive UI Design
- Media URL Validation
- Notification System

# User Roles

## Client

Can

- Create advertisements
- Edit advertisements
- Submit advertisements
- Select promotion packages
- Submit payment proofs
- Track advertisement status

## Moderator

Can

- Review advertisements
- Approve advertisements
- Reject advertisements
- Add moderation comments

## Admin

Can

- Verify payments
- Publish advertisements
- Schedule advertisements
- View analytics dashboard

## Super Admin

Can

- Manage complete system
- Monitor platform activities
- Manage all users

# Advertisement Workflow

   ↓
Submitted
   ↓

   ↓

   ↓
